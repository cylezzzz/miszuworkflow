# **App Name**: FlowCanvasAI

## Core Features:

- Workflow Canvas: A drag-and-drop interface for designing automated workflows.
- Node Registry: Browse and add new nodes (functions) to the workflow from the cloud registry.
- AI-Powered Node Recommendation: The platform will act as a tool suggesting relevant nodes based on the current workflow and task requirements, assisting in workflow creation.
- Workflow Execution: Execute the designed workflow and monitor its progress.
- User Authentication: Login via Google and session simulation for demo mode
- Gallery Sharing: Public and private gallery to showcase your creations. Ratings, required models/nodes and content mode (sfw/nsfw).
- User Profiles and Device Capabilities: Display hardware information to the user. Keep a running history of what they are running in the firestore.

## Style Guidelines:

- Primary color: Deep Indigo (#663399), to invoke sophistication and technological prowess.
- Background color: Very light lavender (#F0EDF5), for a clean and unobtrusive backdrop.
- Accent color: Violet (#8000FF), providing emphasis and contrast without overwhelming the eye.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text. Provides a modern and clean feel while ensuring readability.
- Minimalist line icons to represent different node types and actions, ensuring clarity and a modern aesthetic.
- Three-column layout with a node palette, canvas in the center, and an inspector on the right. Additional "Automation/Flow" view in the center as extra tab/mode. Mimic ComfyUI/n8n feel, with added clarity.
- Subtle animations on node connections and workflow execution, providing visual feedback without being distracting.
